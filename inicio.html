Objetivos del OVI


Objetivo general


  • Desarrollar un sitio web, en el cual se pueda detallar todos los servicios disponibles y el respaldo que ofrece la pagina, la cual estara abierta para el publico en general.

Objetivos especificos


- Creacion de un recurso digital OVI donde se evidencie cada uno de los puntos sobre los cuales va a estar sustentado y justificado el proyecto.


- Comprender los conceptos previos de HTML y CSS.


- Realizar el diseño de la maquetacion con una buena conbinacion de colores


- A travez del grupo colaborativo, aprender a manejar la herramienta GIT y GITHUB para la creacion del sio web.

Objetivos del curso


Objetivo general


Desarrollar competencias para identificar los conceptos fundamentales de las bases de datos.


Objetivos especificos


- Dar a conocer de una forma general qué es y para qué sirve una base de datos, se mostrarán las características de las bases de datos, como se encuentran clasificadas y se mencionaran también los programas o software más utilizados para el desarrollo de las mismas.


- Dar a conocer que es el modelo entidad relación, se explicara que es una entidad, que es una relación (interrelación), que es un atributo y que es un identificador o clave.


- Explicar que es un modelo relacional y cada uno de los conceptos básicos que hacen parte de este modelo como las tablas, las claves, las restricciones de identidad, entre otros.
